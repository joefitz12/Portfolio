ShowSpotter

Concert Ticket Web Application.

The inspiration behind this project was to help our users find out when and where their favorite artists were playing. To achieve this, we built an app that connects with users' Spotify accounts and populates concert info for artists they select. This project utilizes the Spotify and Ticketmaster APIs.

Battle of the Bands

Client-based RPG

This project was inspired by two things that I love very much: 1. RPGs and 2. fictional bands. I built this with the punk aesthetic in mind, so the text is all served up like a DIY sticker pressed on a venue wall. Built with jQuery and BootStrap, all of the elements are displayed dynamically.

SNL Hangman

Client-based Hangman Game

This is a hangman game built in the classic style with a few twists. The theme is SNL, so all the words are names of famous SNL actors, and if you get stuck there's a hint for you. Topped off with the SNL theme song! Built using Vanilla JavaScript and Bootstrap.
